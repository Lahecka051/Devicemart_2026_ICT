# 원고 작성 요령
 아래의 작성 목록을 기준으로 원고를 작성하여 주시기 바랍니다.
- 작성 목록(최대한 자세하게 기술)
1. 작품 제목
2. 작품 개요
3. 작품 설명
    1. 주요 동작 및 특징
    2. 전체 시스템 구성
    3. 개발 환경(개발 언어, Tool, 사용 시스템 등)
4. 단계별 제작 과정
5. 사용한 제품 리스트 (제품명, 제품 링크 포함)
6. 기타(회로도, 소스코드, 참고문헌 등)
