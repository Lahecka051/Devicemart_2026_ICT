# POSTURE AI 시스템 기술 설명서

## 1. 시스템 개요

POSTURE AI 시스템은 사용자의 앉은 자세를 실시간으로 분석하고, 장시간의 자세 패턴을 기록하여 자세 리포트를 생성하는 시스템이다.

**구성 요소**

- Mobile Application
- Raspberry Pi Controller
- STM32 Sensor Node
- Sensor System
- Data Storage 및 Report Engine

센서 데이터는 STM32에서 수집된 후 UART 통신을 통해 Raspberry Pi로 전달되며, Raspberry Pi에서 자세 분류, 자세 점수 계산, 리포트 생성이 수행된다.

---

## 2. 전체 시스템 구조
```
Mobile Application
        │
        │ WiFi (HTTP / WebSocket)
        ▼
Raspberry Pi Controller
        │
        │ UART Communication
        ▼
STM32 Sensor Node
        │
        ▼
Seat Sensors / ToF Sensors / IMU
```

### 구성 요소별 역할

**Mobile Application**
- 사용자 인터페이스 제공
- 사용자 프로필 등록 및 선택
- 측정 시작 / 중지 / 재측정 결정
- 실시간 자세 상태 확인
- 리포트 확인

**Raspberry Pi Controller**
- STM32와 UART 통신 수행
- 센서 데이터 수신 및 해석
- 자세 분류 알고리즘 수행
- 자세 점수 계산 및 리포트 생성
- 모바일 앱과 HTTP / WebSocket 통신

**STM32 Sensor Node**
- 센서 데이터 수집
- 착석 여부 판단
- 사용자 자리 이탈(STAND) 이벤트 감지
- 센서 데이터를 Raspberry Pi로 전송

## 5. 측정 동작 흐름
```
1. Raspberry Pi 부팅
2. STM32 UART handshake 수행
3. 사용자 프로필 등록 또는 선택
4. 캘리브레이션 수행 (필요 시)
5. 측정 시작 여부 확인
6. 착석 여부 확인
7. 측정 시작
8. 실시간 자세 분석 수행
9. STAND 이벤트 감지
10. 재측정 여부 결정
11. 세션 종료 및 리포트 저장
```

---

## 6. UART 통신 프로토콜

### Raspberry Pi → STM32

| 메시지 | 설명 |
|--------|------|
| `CHK_SIT` | 착석 여부 확인 |
| `CAL` | 캘리브레이션 시작 |
| `GO` | 측정 시작 |
| `STOP` | 측정 일시 중지 |
| `QUIT` | 측정 완전 종료 |

### STM32 → Raspberry Pi

| 메시지 | 설명 |
|--------|------|
| `READY` | STM32 부팅 완료 |
| `LINK_OK` | UART 연결 확인 |
| `SIT` | 사용자 착석 상태 |
| `STAND` | 사용자 자리에서 일어남 |

---

## 7. STAND 이벤트 처리

STM32는 사용자가 자리에서 일어난 경우 `STAND` 이벤트를 Raspberry Pi로 전송한다.

**처리 절차**

1. Raspberry Pi가 `STAND` 이벤트 수신
2. Mobile App에 자리 이탈 알림 전달
3. 사용자에게 재측정 여부 선택 요청

**재측정 선택 시** → `CHK_SIT → SIT → GO` 절차로 측정 재개

**재측정 거부 시** → 세션 종료 및 리포트 저장

---

## 8. 측정 중단 정책

### 측정 일시 중지 (잠시 자리를 비우는 경우)
```
RPi → STM32 : STOP
```

STM32는 측정을 중단하고 idle 상태로 대기한다.

### 측정 완전 종료 (오늘 측정을 끝내는 경우)
```
RPi → STM32 : QUIT
```

STM32는 내부 상태를 정리하고 idle 상태로 전환된다.

## 9. 데이터 저장 구조

SQLite 데이터베이스를 사용하며, 주요 테이블 구조는 다음과 같다.

| 테이블 | 설명 |
|--------|------|
| `users` | 사용자 프로필 정보 |
| `baselines` | 캘리브레이션 기준값 |
| `sessions` | 측정 세션 기록 |
| `minute_reports` | 분 단위 자세 리포트 |
| `daily_reports` | 일일 자세 요약 리포트 |
